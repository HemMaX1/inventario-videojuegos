# Inventario de Videojuegos 🎮

Sistema de inventario interactivo hecho en Python. Día 5 de mi ruta para aprender a programar.

### Funciones principales
1. **Ver inventario** - Lista todos los juegos numerados con `for` + `enumerate()`
2. **Agregar juego** - Añade nuevos títulos al inventario con `.append()`
3. **Borrar juego** - Elimina juegos de la lista con `.remove()`
4. **Validación** - Revisa si la lista está vacía con `len()`

### Conceptos de Python aplicados
- Listas `[]` para almacenar colecciones de datos
- Ciclo `for in` para recorrer elementos
- `enumerate()` para obtener índice + valor
- Métodos de lista: `.append()` y `.remove()`
- `len()` para contar elementos
- `while True` para menú infinito
- `f-strings` para imprimir variables
- Condicionales `if/elif/else`
